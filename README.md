# Bike Sales Dashboard | Microsoft Excel
Developed an interactive Bike Sales Dashboard in Microsoft Excel using Pivot Tables, Pivot Charts, Slicers, and data visualization techniques. The dashboard enables dynamic analysis of customer purchasing behavior based on demographic and lifestyle attributes.
# Features
Interactive dashboard with dynamic filtering using Slicers
Pivot Tables and Pivot Charts for data summarization
Analysis of average income by purchase status
Customer age bracket analysis
Customer commute distance analysis
Dashboard filters for Marital Status, Region, and Education
Clean and professional dashboard design
# Tools & Skills Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Data Cleaning
Data Analysis
Dashboard Design
Data Visualization
# Key Insights
Compared average income of customers who purchased bikes versus those who did not.
Analyzed bike purchases across different age groups.
Examined the impact of commute distance on bike purchases.
Enabled interactive filtering for deeper business insights.
# Project Outcome
Built a fully interactive Excel dashboard that transforms raw sales data into meaningful business insights, helping stakeholders explore customer purchasing patterns through dynamic visualizations.
